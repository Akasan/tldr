# kdash

> A simple terminal dashboard for Kubernetes.
> Mode information: <https://github.com/kdash-rs/kdash/>.

- Show dashboard:

`kdash`

- Show dashboard in debug mode and write logs:

`kdash {{[-d|--debug]}}`

- Set the tick rate:

`kdash {{[-t|--tick-rate]}} 100`

- Set the polling rate:

`kdash -p 200`
